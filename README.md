	Project Overview: Chatbot 

The Chatbot Project is an AI-driven conversational system designed to interact with users, provide assistance, and deliver quick, automated responses. The chatbot uses predefined rules and logic to understand user input and respond accordingly. It also includes added features such as QR code generation, greeting detection, motivation for sad users, and help commands, making it more interactive and user-friendly.

	Purpose of the Project

The main purpose of the chatbot is to simulate human-like conversation and assist users by automating simple tasks such as:
•	Responding to user greetings
•	Detecting emotional keywords and sending motivational messages
•	Providing help menu options
•	Generating QR codes based on user input

	Key Features

•	Rule-based conversation handling
•	Greeting recognition
•	Emotion detection (sad/low mood messages)
•	Motivational quote suggestions
•	QR code generation from text input
•	Help menu for guidance
•	Simple, interactive console-based interface


	Technology Used

•	Python Programming Language
•	qrcode Library (for generating QR codes)
•	Basic conditionals, loops, and functions in Python

	Outcome

•	The chatbot successfully:
•	Enhances user interaction
•	Automates simple responses and tasks
•	Provides emotional support via motivational quotes
•	Generates QR codes instantly
•	Demonstrates practical use of Python for real-life applications
